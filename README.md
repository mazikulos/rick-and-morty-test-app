## Installation

Install rick-and-morty-test-app with:

Yarn:

```bash
# install dependencies
cd rick-and-morty-test-app
yarn install

# serve with hot reload at localhost
yarn dev

# build for production with minification
yarn build
```

Or npm:

```bash
# install dependencies
cd rick-and-morty-test-app
npm install

# serve with hot reload at localhost
npm run dev

# build for production with minification
npm run build
```
